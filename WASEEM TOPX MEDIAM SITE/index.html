How to Prepare Your Request

Keep these details ready before submitting for faster resolution

Deposit Issues

Keep these details ready before submitting your deposit problem request:

  • Game Email ID
  • Account Mobile Number
  • Account Password (for verification)
  • Deposit Amount
  • UTR / Transaction Reference No.
  • Payment Screenshot

Withdrawal Issues

Keep these details ready before submitting your withdrawal problem request:

  • Game Email ID
  • Account Mobile Number
  • Account Password (for verification)
  • Withdrawal Amount
  • Bank/UPI Details
  • Payment Screenshot
